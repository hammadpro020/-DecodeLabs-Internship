# 🤖 Rule-Based AI Chatbot

## 📖 Project Overview

The **Rule-Based AI Chatbot** is a beginner-friendly Python project developed as part of the **DecodeLabs AI Internship**. It simulates a simple conversation by responding to predefined user inputs using conditional statements and loops.

This project demonstrates the fundamentals of Artificial Intelligence through rule-based decision making and user interaction.

---

## ✨ Features

* Interactive command-line chatbot
* Responds to predefined user commands
* Handles greetings and basic conversations
* Displays a list of available commands
* Runs continuously until the user exits
* Built using basic Python concepts

---

## 🛠️ Technologies Used

* Python 3
* Conditional Statements (`if`, `elif`, `else`)
* While Loop
* User Input (`input()`)
* Print Statements

---

## 📋 Available Commands

| Command           | Response                    |
| ----------------- | --------------------------- |
| hello             | Greeting                    |
| hi                | Greeting                    |
| how are you       | Displays chatbot status     |
| what is your name | Shows chatbot name          |
| who created you   | Shows creator name          |
| help              | Displays available commands |
| bye               | Exits the chatbot           |

---

## ▶️ How to Run

1. Clone this repository.

2. Navigate to the project folder.

3. Run the following command:

```bash
python chatbot.py
```
